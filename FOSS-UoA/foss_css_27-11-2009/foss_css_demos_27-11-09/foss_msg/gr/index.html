by SpartanGeorge  on  24 / 11 / 2009

Βρισκόμαστε στην ευχάριστη θέση, να σας πληροφορήσουμε ότι την Παρασκευή 27 Νοεμβρίου καθώς και την Παρασκευή 04 Δεκεμβρίου, ώρα 13.00 μέχρι 16.00 περίπου, στην αίθουσα Α2 (τμήμα Πληροφορικής και Τηλεπικοινωνιών), θα διοργανωθούν τα δύο μέρη της παρουσίασης της κοινότητας σχετικά με τεχνολογίες για web development.

Το πρώτο μέρος της παρουσίασης θα ασχοληθεί με HTML 4.0 (XHTML 1.0), με CSS 2.0 και με μία μικρή εισαγωγή στη PHP.
