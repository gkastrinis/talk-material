Στο αρχείο /etc/apache2/sites-available/default μπορούμε να ορίσουμε το σημείο που θα κοιτάει ο apache για τα web αρχεία μας.

Αν σε έναν browser δώσουμε την διεύθυνση localhost, το παραπάνω σημείο είναι το μέρος που θα κοιτάξει ο apache για web σελίδες κτλ.

Για παράδειγμα αν ορίσουμε σαν DocumentRoot (και στο Directory αντίστοιχα) το /home/george/web και μέσα εκεί υπάρχει ένα αρχείο hello.php, μπορούμε να ανοίξουμε το αρχείο αυτό από τον browser μας, απο την διεύθυνση localhost/hello.php


Στο αρχείο /etc/php5/apache2/php.ini μπορούμε να αλλάξουμε διάφορες ρυθμίσεις, σχετικές με την php. Μία χρήσιμη ρύθμιση είναι η error_log. Στη ρύθμιση αυτή δηλώνουμε σε ποιό αρχείο θα γράφονται τα διάφορα errors και warnings που τυχών συμβαίνουν στον κώδικα μας.

Για παράδειγμα αν έχουμε την ρύθμιση error_log = /var/log/php.log
δημιουργήσουμε το αρχείο /var/log/php.log (sudo touch /var/log/php.log)
και του δώσουμε δικαιώματα για εγγραφή (sudo chmod a+w /var/log/php.log)
τότε κάθε φορά που συμβαίνει κάποιο error ή κάποιο warning, θα καταγράφεται στο παραπάνω αρχείο.


Σημείωση: κάθε φορά που κάνουμε κάποια αλλαγή στα παραπάνω αρχεία, πρέπει να επανεκκινούμε τον apache για να ισχύσουν οι αλλαγές μας. Αυτό γίνεται με την εντολή sudo /etc/init.d/apache2 restart
